new ZhenshangyinTimePicker('inputSelector', options);
参数 类型 默认值 说明
type string 'date' 选择器类型，可选值：'date'、'dateRange'、'week'、'month'、'monthRange'、'year'、'yearRange'、'time'、'timeRange'
dateFormat string - 日期/时间的显示和输出格式，如 'YYYY-MM-DD'，默认根据 type 自动设置
language string 'zh' 语言设置，可选值：'zh'（中文）、'en'（英文）
separator string - 范围选择或多选时的分隔符
multiSelect boolean false 是否允许多选，仅适用于 date、month、year 类型
dateDelimiter string ',' 多选时各个日期之间的分隔符
availableHours number[] [0-23] 可选的小时数组，限制可选小时范围，默认 0-23
availableMinutes number[] [0-59] 可选的分钟数组，限制可选分钟范围，默认 0-59
availableSeconds number[] [0-59] 可选的秒数组，限制可选秒范围，默认 0-59
disabledWeekdays number[] [] 禁用的星期几，0=周日，6=周六，如 [0, 6] 表示禁用周末
disabledItems array [] 禁用的具体项目（日期、月份或年份），支持格式：'YYYY-MM-DD'（日期）、'YYYY-MM'（月份）、数字年份，如 ['2024-01-01', '2024-01', 2022]
disabledRanges array [] 禁用的范围（日期、月份或年份区间），格式：[{start: '2024-01-01', end: '2024-01-07'}, {start: '2024-06', end: '2024-08'}, {start: 2020, end: 2022}]
onSelect function - 选择完成后的回调函数，参数 value 为选择的值

参数 类型 默认值 说明
direction string 'horizontal' 轮播方向，可选值：'horizontal'（水平）、'vertical'（垂直）
effect string 'slide' 切换效果，可选值：'slide'（滑动）、'fade'（淡入淡出）
loop boolean false 是否开启循环播放
speed number 300 切换动画时长（毫秒）
autoplay boolean false 是否自动播放
interval number 3000 自动播放间隔时间（毫秒）
pauseOnMouseEnter boolean false 鼠标悬停时是否暂停自动播放
slidesPerView number/string 1 同时显示的 slide 数量
spaceBetween number 0 slide 之间的间距（像素）
slidesPerGroup number 1 每次切换的 slide 数量
centeredSlides boolean false 是否居中显示当前 slide
linkedGroup string - 联动轮播配置，可定义值
clickToSlide boolean true 是否启用点击切换功能
breakpoints object - 响应式断点配置，根据屏幕宽度设置不同参数

touchable boolean true 是否启用触摸滑动
magnifier boolean false 是否开启放大镜功能
fixedMagnifierSize number - 固定位置放大镜的尺寸（像素）
showSelectionBox boolean true 是否显示选区框
selectionBoxSize number - 选区框的大小（像素），默认根据放大倍率自动计算
magnifierMargin number - 放大镜的间距（像素）
zoomRatio number - 放大倍率
perspective number 1500 3D 变换的视距（像素）
transforms object - 自定义轮播图样式配置，包含 scale（缩放）、translate（平移）、rotate（旋转）、skew（斜切）、perspective（透视）、backfaceVisibility（背面可见性）、opacity（透明度）、filter（滤镜）、uniformSpacing（统一可视间距）等配置
