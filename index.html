Message 消息提示

new ZhenshangyinMessage({ });
参数 类型 默认值 说明
message string 无 通知显示的主要内容。
type string 'info' 通知的类型，决定通知的背景颜色和图标。可选值：'info'、'success'、'warning'、'error'。
duration number 2000 通知显示的时长（以毫秒为单位）。通知在此时间后自动消失，若为 0 或未指定则通知永不消失。

Notification 通知

new ZhenshangyinNotification({ });
参数 类型 默认值 说明
title string 无 通知的标题文本，显示在通知的顶部部分。
message string 无 通知的主要内容，显示在通知的正文部分。
type string 'info' 通知的类型，决定通知的背景颜色和图标。可选值：'info'、'success'、'warning'、'error'。
duration number 2000 通知显示的时长（以毫秒为单位），到达此时间后通知自动消失。若为 0 或未指定则通知永不消失。

new ZhenshangyinDatePicker('inputSelector', {});
参数 类型 默认值 说明
inputSelector string null 选择日期输入框的 CSS 选择器。
dateFormat string 'YYYY-MM-DD' 日期格式，支持多种格式，如：'YYYY-MM-DD', 'MM/DD/YYYY', 'DD/MM/YYYY'等。
onDateSelect function () => {} 日期选择后的回调函数。

Select 选择器

  • 下拉选择

  • 下拉多选带删除

  • 下拉多选

new ZhenshangyinDropdown({ });
参数 类型 默认值 说明
container string - 必填，用于指定下拉菜单挂载的容器的选择器字符串。
data array - 必填，下拉菜单的数据项数组。
customParams object {} 可选，自定义参数对象。
onSelect function () => {} 可选，当选项被选中时的回调函数。
searchEnabled boolean false 可选，是否启用搜索功能。
inputSearchEnabled boolean false 可选，是否在输入框中启用搜索功能。
defaultSelected string/array null 可选，初始选中的选项标题或标题数组（多选时）。
grouped boolean false 可选，数据项是否分组。
multiSelect boolean false 可选，是否允许多选。
inputMultiSelect boolean false 可选，是否在输入框中显示多选结果。

Cascader 级联选择器

  • 省市区级联选择

  • 无限联动

  • 省市区级联选择

new ZhenshangyinLinkage({ });
参数 类型 默认值 说明
container string - 级联选择器的容器选择器，例如 #cascaderProvinceCityDistrict。
multiInputs string[] - 多输入框的选择器数组，例如 ['#cascaderProvinceInput', '#cascaderCityInput', '#cascaderDistrictInput']。
data object[] - 数据源，包含级联选择的数据结构。
separator string "-" 分隔符，用于连接选择的值，默认为 -。
callback function () => {} 回调函数，在选择值发生变化时触发，参数为选中的值。
